# 📊 Daily Price 日均价格

> 极简、专注于「日均成本」计算的个人资产管理工具。

---

## ✨ V2.0 核心特性

### 📊 动态日均算法
- **三态状态机**：支持「服役中 / 已退役 / 已卖出」三种资产状态
- **智能冻结逻辑**：资产退役或卖出后，时间自动冻结在出售/退役日期，不再继续折旧
- **预期寿命计算**：服役中的资产可设定预期使用天数，在寿命期内按固定天数平摊成本
- **回血抵扣**：卖出时可记录回血价，自动从总成本中扣除后计算实际日均

### 📱 现代交互设计
- **悬浮岛导航栏**：底部毛玻璃特效胶囊导航，iOS 风格悬浮 Dock
- **双列瀑布流橱窗**：资产以紧凑网格卡片呈现，信息密度高且美观
- **状态可视化**：服役中(绿色)、已退役(灰色)、已卖出(红色)一目了然
- **置顶优先级**：重要资产可置顶，始终排在列表最前

### 🖼️ 本地图像引擎
- **相册选图**：直接从系统相册选择资产图片
- **1:1 正方形裁剪**：集成 `image_cropper` 实现精确裁剪
- **脱机保存**：图片保存至应用文档目录，无需网络，极度轻量
- **首字母头像**：无图片资产自动生成彩色首字母头像

### 🗄️ 工业级存储
- **SQLite 底层**：采用极其稳定的 `sqflite` 作为数据引擎
- **UUID 主键**：全表使用 UUID v4 作为主键，支持安全的数据导入导出
- **批量 Upsert**：导入数据时智能查重，支持插入/更新混合操作
- **大数据量支持**：轻松应对数千条资产记录的极速读写

## 🚀 快速运行

```bash
# 克隆项目
git clone https://github.com/cecilenn/Daily-Price.git
cd Daily-Price

# 安装依赖
flutter pub get

# 运行（开发模式）
flutter run

# 构建 Android Release
flutter build apk --release
```

### 环境要求
- Flutter SDK: ^3.11.0
- Dart SDK: ^3.11.0
- Android SDK: API 21+
- iOS: 12.0+
